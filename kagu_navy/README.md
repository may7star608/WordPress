# 家具・雑貨専門店向け

TEMPLATE PARTY様の無料テンプレート「家具・雑貨専門店向け 無料ホームページテンプレート tp_kagu1_black」をWordPress化しました。

## ポイント
ナビゲーションメニューや各セクションのリンクからそれぞれのページへ遷移できるように<br>
CPT UIを使って商品一覧、おすすめ商品のカスタム投稿ページを作成

### 商品一覧
ソファ、ダイニング、インテリア雑貨の3つの商品カテゴリーを選択できるようにし、各カテゴリは最大6件まで表示できるように、ループ処理させました。

### おすすめ商品
最大9件まで表示できるように、ループ処理させました。

### お知らせ
標準の投稿から日付と投稿内容をクライアント側で変更できるようなっています。
